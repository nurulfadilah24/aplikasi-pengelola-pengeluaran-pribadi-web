# UI/UX Design Project ‚Äì Aplikasi Pengelola pengeluaran pribadi

## Deskripsi Project
Proyek ini merupakan tugas pembuatan *UI/UX Design* untuk aplikasi pengelola pengeluaran pribadi.  
Tujuan utama dari project ini adalah membuat tampilan antarmuka (UI) dan pengalaman pengguna (UX) yang menarik, sederhana, serta mudah digunakan oleh pengguna.

Selain itu, dibuat juga *timeline (Gantt Chart)* menggunakan *ClickUp* untuk membantu mengatur jadwal dan proses pengerjaan setiap tahapan.

---

## Komponen Project
Proyek ini dibagi menjadi empat bagian utama:

### 1. Storyboard Project
- Menampilkan alur cerita atau perjalanan pengguna (user flow) di dalam aplikasi.  
- Menggambarkan skenario interaksi pengguna dari membuka aplikasi hingga melakukan aktivitas utama seperti login atau melihat produk.
<Img src="gambar Storyboard.png">

### 2. Mockup Project
- Menunjukkan tampilan awal atau desain visual dasar dari aplikasi.  
- Berfokus pada pemilihan warna, tata letak elemen, serta struktur dasar setiap halaman.  
- Contoh: tampilan halaman splash screen dan login.
<Img src="gambar mockup.png">

### 3. UI (User Interface) Project
- Merupakan hasil desain antarmuka final dari aplikasi.  
- Menampilkan elemen lengkap seperti tombol, ikon, navigasi, serta tata letak visual.  
- Dibuat menggunakan *Figma* agar dapat diatur secara konsisten di semua halaman.
<Img src="gambar UI.png">

## 4. UX (User Experience) Project
- Menguji dan menilai bagaimana pengguna berinteraksi dengan aplikasi.  
- Meliputi analisis kenyamanan navigasi, kemudahan penggunaan, serta tampilan keseluruhan.  
- Diuji menggunakan fitur prototype di Figma.
